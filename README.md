# 🎂 Happy Birthday Interactive Website

A beautiful, interactive birthday celebration website featuring slide-based navigation, animations, and interactive elements. Built with pure HTML, CSS, and JavaScript.


## ✨ Features

### 🎯 Core Experience
- **Slide-based Navigation** - No scrolling required, navigate through 12 beautifully designed slides
- **Auto-play Mode** - Automatic slide advancement with manual override
- **Keyboard Navigation** - Use arrow keys, space, or enter to navigate
- **Touch/Click Support** - Click anywhere to advance (with smart element detection)

### 🎨 Visual Effects
- **Floating Hearts** - Mouse trail with emoji hearts that follow your cursor
- **Particle System** - Background floating hearts animation
- **Fireworks** - Click anywhere to create fireworks, auto-trigger on celebration
- **Confetti** - Colorful confetti explosions on special moments
- **Balloons** - Floating balloon animations

### 🎵 Audio
- **Background Music** - Auto-playing synthesized melody using Web Audio API
- **Music Toggle** - Play/pause button with animated visualizer

### 🎂 Interactive Elements
- **Blow Out Candles** - Click the cake to blow out candles one by one
- **Flip Cards** - Click wish cards to reveal heartfelt messages
- **Photo Gallery** - Click images to open lightbox view
- **Typewriter Text** - Animated text reveals in chat and letter sections

### 📱 Responsive Design
- Fully responsive layout for desktop, tablet, and mobile devices
- Optimized touch interactions for mobile

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools required - runs directly in browser
